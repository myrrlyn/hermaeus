# Hermaues Changelog

## v1

### v1.0.0

Added a storage backend (`Apocryphon` and `Archivist` classes) capable of
formatting the retrieved text and storing it on disk.

`mora` is confirmed to work in the wild, and so Hermaeus is ready for a 1.0
release.
